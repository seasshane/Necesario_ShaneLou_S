1. Git Push Workflow: Necesario_ShaneLou_S

REM Step 1: Open your project folder in Terminal
cd %USERPROFILE%\OneDrive\Desktop\Necesario_ShaneLou_S

REM Step 2: Configure Git identity (only once per computer)
git config --global user.name "Budjay Necesario"
git config --global user.email "seasshane@gmail.com"

REM Step 3: Initialize Git repository (only if not already initialized)
git init

REM Step 4: Add all project files to staging
git add .

REM Step 5: Commit your files with a message
git commit -m "first commit"

REM Step 6: Rename your branch to 'main' (GitHub default)
git branch -M main

REM Step 7: Add your GitHub repository as remote
git remote add origin https://github.com/seasshane/Necesario_ShaneLou_S.git

REM Step 8: If remote already exists, update it
git remote set-url origin https://github.com/seasshane/Necesario_ShaneLou_S.git

REM Step 9: Push your project to GitHub
git push -u origin main

REM Step 10: Verify on GitHub
REM Open: https://github.com/seasshane/Necesario_ShaneLou_S
